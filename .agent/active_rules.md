# Active Rules

## Universal Rules
*   [ ] No Silent Failures: If a command fails, STOP and investigate.
*   [ ] Logs are Truth: If it's not in the log, it didn't happen.
*   [ ] Tests are Law: You cannot mark a task complete if tests are failing.
*   [ ] Lovable Handoff: If a change requires Database or Environment updates, you MUST generate a prompt for the user for Lovable.
