---
description: How to implement a new feature
---
1.  **Requirement**
    *   Update `project_requirements_document.md`.
    *   Add to `enhancement_log.md`.
2.  **Plan**
    *   Create implementation plan.
3.  **Test**
    *   Write feature tests.
4.  **Implement**
    *   Write code.
    *   Pass tests.
5.  **Verify**
    *   Manual verification if needed.
    *   Update logs.
